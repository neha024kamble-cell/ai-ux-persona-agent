AI User Persona Architect (UX Research Automation)

An agentic AI workflow built for UI/UX Designers to automate the synthesis of qualitative user research into high-fidelity user personas.

Why This Matters

Synthesizing raw interview notes is a major bottleneck in the UX process. This tool uses Agentic AI to:

Reduce Bias: Systematically identifies themes across conversational data.

Save Time: Converts hours of manual synthesis into seconds of automated processing.

Ensure Privacy: Runs 100% locally via Ollama, ensuring sensitive user data never leaves your machine.

🛠️ The "Agent" Logic

This project uses CrewAI to orchestrate a specialized multi-agent workflow:

Research Analyst Agent: Extracts pain points and behaviors from messy notes.

Persona Architect Agent: Synthesizes these insights into a structured, empathetic persona document.

💻 Tech Stack

Python 3.12

CrewAI (Agent Orchestration)

Ollama / Llama 3 (Local LLM)

Streamlit (User Interface)
